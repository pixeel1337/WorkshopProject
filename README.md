# Workshop Manager - Application 🚗🛠️

Część aplikacyjna projektu **WorkshopProject**, stanowiąca rdzeń systemu do zarządzania warsztatem samochodowym. Narzędzie zostało zaprojektowane, aby usprawnić codzienną pracę mechaników oraz obsługę klienta.

## 🌟 Kluczowe Funkcjonalności
- **Zarządzanie Zleceniami**: Rejestrowanie nowych napraw, opisywanie usterek i monitorowanie postępów prac.
- **Baza Klientów i Pojazdów**: Kompleksowa ewidencja właścicieli wraz z historią serwisową ich samochodów.
- **Harmonogram Pracy**: Planowanie terminów wizyt i zarządzanie obłożeniem stanowisk warsztatowych.
- **Statusy Napraw**: Czytelna informacja o etapie prac (np. "Oczekuje", "W trakcie", "Gotowe do odbioru").
